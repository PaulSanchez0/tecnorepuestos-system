# tecnorepuestos-system
# TecnoRepuestos S.A.

Sistema web para gestión de inventario, compras, ventas y portal de clientes.

---

## Tecnologías utilizadas

- React JS
- Node.js
- Express
- MySQL

---

## Estructura del proyecto

```bash
src/
routes/
controllers/
services/
models/
middlewares/
config/
db/
tests/
public/
docs/
```

---

## Descripción de carpetas

### src/
Contiene el código fuente principal del sistema.

### routes/
Contiene las rutas de la API REST.

### controllers/
Contiene la lógica de controladores.

### services/
Contiene las reglas de negocio.

### models/
Contiene modelos de datos.

### middlewares/
Contiene validaciones y autenticación.

### config/
Configuraciones generales del sistema.

### db/
Configuración de base de datos.

### tests/
Pruebas unitarias e integración.

### public/
Archivos públicos y recursos estáticos.

### docs/
Documentación técnica del proyecto.

---
